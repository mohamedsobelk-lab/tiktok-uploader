# Private TikTok Video Uploader

This is a desktop application that allows authenticated users to upload videos to TikTok using official API integration.

## Features
- Upload videos to TikTok
- Local processing (no data storage)
- Secure authentication via TikTok API

## Privacy
This application does not store or share user data.

## Usage
This tool is intended for personal content management.
